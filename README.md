# 30 DAY TERRAFORM CHALLENGE

This repository documents my progress through a 30-day sprint to learn Terraform and Infrastructure as Code (IaC).

The goal of this challenge is to build a strong foundation in Terraform by learning the concepts, completing hands-on labs, and documenting the journey day by day.

Each day includes:

- Notes from learning resources
- Hands-on Terraform exercises
- Infrastructure experiments
- Reflections and key takeaways

This challenge is sponsored by:

- **HashiCorp**
- **EveOps**
- **AWS AI/ML User Group Kenya (Main)**

The objective is to gain practical experience with Terraform and understand how Infrastructure as Code is used to build and manage modern cloud infrastructure.
